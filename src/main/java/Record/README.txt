Java Record
==============

Java Record — особый вид класса Java, который
имеет краткий синтаксис для определения неизменяемых
классов только для данных.

Экземпляры Java Record могут быть полезны для хранения
записей, возвращенных из запроса к базе данных,
записей, возвращенных из удаленного вызова службы,
записей, прочитанных из CSV-файла, или аналогичных
типов вариантов использования.

Java Record состоит из одного или нескольких полей данных,
которые соответствуют переменным-членам в обычном классе Java.
Компилятор Java автоматически генерирует методы getters, toString(),
hashcode() и equals() для этих полей данных, поэтому не нужно
самостоятельно писать этот шаблонный код.

Поскольку Java Record неизменяем, методы setters не создаются.

Java Record были введены с намерением использовать их в качестве
быстрого способа создания классов носителей данных, т. е. классов,
целью которых является простое хранение данных и перенос их между
модулями, также известных как POJO (Plain Old Java Objects) и
DTO (Data Transfer Objects).

Определение типа Record является final, то есть не можете создавать
подклассы типа Java Record.


Java Record введен в Java SE 14 в качестве функции предварительного
просмотра, которая представляет собой функцию, дизайн, реализация и
спецификация которой завершены, но она не является постоянным дополнением
к языку, что означает, что эта функция может существовать или не существовать
в будущих версиях. языка.

Java SE 15 расширяет функцию предварительного просмотра дополнительными
возможностями, такими как локальные классы Record.

В Java 16 Record стал стандартной функцией JDK.
