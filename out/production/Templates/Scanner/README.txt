Java. Класс Scanner
=====================
Java документация:
https://docs.oracle.com/javase/8/docs/api/java/util/Scanner.html

Простой сканер текста, который может анализировать примитивные
типы и строки с помощью регулярных выражений.

Scanner разбивает свой ввод на токены, используя шаблон разделителя,
который по умолчанию соответствует пробелам. Затем полученные токены
можно преобразовать в значения разных типов с помощью различных методов next.

Используется для чтения данных из входящих потоков (консоль приложения, файл и т.д.)
